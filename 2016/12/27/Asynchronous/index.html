    异步

    发表于 2016-12-27 | 分类于 Javascript | | 阅读次数
    通俗眼光看异步

    一、setTimeout

    1.setTimeout基本用法

    2.setTimeout只会推迟执行，而不会提前执行

    3.setTimeout有delay的最小值

    4.setTimeout模拟setInterval

    二、一个简单的动画

    5.setTimeout特点

    • setTimeout是浏览器提供的API
    • setTimeout里面的this默认指向window;可以通过bind来改变this的指向，例子
    • timer id 就是第几个闹钟

    三、异步

    1. 表面的结论

    代码的执行顺序和写代码的顺序不一致，“很可能”是异步。

    2. 异步就是一种套路

    • 例子： 托人买票
      你让黄牛给你买票，你是一直追着他要到票为止，还是拜托他买到票后call me back，自己在这段时间就去干别的事情。毫无疑问后者就是异步
    • 异步不等于callback，只是JS中异步和callback常常一起出现
    • 异步就是猴急，不等结果
    • 同步就是耐心，等结果

    四、callback

    异步没有立即返回的结果

    • setTimeout 的结果是一个id
    • $.get返回的是一个请求对象
    • addEventListener的结果是空

    想要拿到异步的结果就要通过callback来实现；

    这里是你的幸福之源
